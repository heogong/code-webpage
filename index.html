CODE

CODE는 국내 대표 가상자산 거래소 3사(빗썸, 코인원, 코빗)가 2022년 예정된 트래블룰 시행에 앞서 설립한 국내 첫 VASP 컨소시엄입니다.

2021년 8월 출범 이후, 금융당국과 국제자금세탁방지기구(FATF)의 기준에 부합하는 솔루션 개발을 통해 글로벌 트래블룰 표준화 방안을 선도적으로 제시하고자 노력해왔습니다.

2021년 12월, CODE 기술력으로 완성한 블록체인 기반의 선진적, 혁신적 트래블룰 솔루션 서비스를 시작합니다.






국내 VASP 파트너 및 연구개발 파트너

블록체인 기반의
 한국형 트래블룰 솔루션

  • 80여개 글로벌 금융사가 참가하는 R3 블록체인 컨소시엄
  • 금융 서비스에 최적화된 분산원장기술 서비스 제공
  • 상호 허가된 기관만 접속 가능한 Permissioned Blockchain
Corda 노드간 데이터베이스 공유 구조
  • 효율성

    중개자가 없는 블록체인 기반 시스템: 수수료 등 비용 절감 및 빠른 데이터 처리

  • 안전성

    각 노드는 직접 연관된 데이터만 보유: 개인정보의 안전한 교환 및 저장

  • 확장성

    확장성 위주의 시스템 디자인: 네트워크 신규 가입 및 타 네트워크 연계 용이

  • 편의성

    고객 경험을 최우선으로 두는 Address Search System 도입
