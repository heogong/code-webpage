ABOUT CODE

CODE는 가상자산거래소 3사(빗썸,코인원,코빗)가 공동설립한 대한민국 유일 트래블룰 컨소시엄입니다. 국제자금세탁방지기구(FATF) 및 규제기관의 기준에 부합하는 글로벌 트래블룰 표준을 준수하여 VASP에 특화된 기술 솔루션과 맞춤형 컴플라이언스 서비스를 제공합니다.

VASP와 트래블룰이란?

Virtual Asset Service Provider의 약자로 가상자산을 매개로 한 사업을 영위하는 가상자산 사업자를 의미합니다.

표적으로 가상자산거래소, 가상자산수탁사업자, 지갑사업자 등이 해당됩니다.

Q. A.

CODE 솔루션의 장점은 무엇인가요?

CODE는 국내 대표 가상자산거래소 3사(빗썸, 코인원 코빗)가 설립한 대한민국을 대표하는 최초의 트래블룰 컨소시엄입니다. 2021년 8월 출범 이후, 금융당국과 국제자금세탁방지기구(FATF)의 기준에 부합하는 솔루션 서비스를 통해 글로벌 트래블룰 표준을 제시해 오고 있습니다.
